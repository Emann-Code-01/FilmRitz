# FilmRitz — A Netflix-inspired Movie Explorer

FilmRitz is a lightweight, movie discovery app built with Vue.js and Tailwind CSS. It uses The Movie Database (TMDB) API to surface trending movies, details, and images so users can browse and discover films with a modern UI.

Live link: https://filmritz.vercel.app/

## Key features

- Trending movies feed (TMDB)
- Movie detail pages with poster, overview, rating, genres, and trailers
- Responsive, modern UI powered by Tailwind CSS
- Built with Nuxt 3 for server-side rendering and optimal performance
- Simple, easy-to-extend code structure — great for learning Nuxt + TMDB integration
- etc. (https://filmritz.vercel.app/)

---

## Tech stack

- (Vue 3)
- Tailwind CSS
- Fetch + TMDB REST API
- Optional: Pinia (state), composables for API logic

---

## Prerequisites

- Node.js 18+ (or LTS)
- A TMDB API key: https://www.themoviedb.org/documentation/api
